# AW Chemistry LMS — UI/UX Design

A modern Learning Management System designed for chemistry education, enabling students and administrators to manage classes, quizzes, and academic performance through an intuitive digital platform.

## 📖 About the Project

AW Chemistry LMS is a UI/UX design project created for an advanced chemistry learning platform focused on improving online education experiences for both students and administrators. The system provides a streamlined environment where students can easily access learning materials, participate in quizzes, and monitor their academic progress.

The platform also includes a powerful admin management system that allows administrators to create and manage classes, handle quizzes, and update student results efficiently. The design emphasizes simplicity, accessibility, and a clean educational interface to enhance user engagement and usability.

## ✨ Key Features

* 🔐 User Authentication — Secure student and admin registration & login system
* 👨‍🎓 Student Dashboard — Access classes, quizzes, and examination results
* 🧪 Online Quizzes — Participate in chemistry quizzes through the platform
* 📊 Result Management — Students can view updated quiz and class results
* 🏫 Class Management — Admins can create, update, and delete classes
* 📝 Quiz Management — Admins can manage quizzes and assessment content
* 📈 Performance Tracking — Organized academic progress monitoring system
* 🎨 Modern UI/UX — Clean and responsive educational interface design
